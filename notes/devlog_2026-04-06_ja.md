  - cold: `0-3072`
  - hot: `3072-4096`
  - blocks: `24`
- `seq_len=8192`
  - cold: `0-7168`
  - hot: `7168-8192`
  - blocks: `56`

## 今日詰まったこと

- `python scripts/run_tiered_kv.py` で
  `ModuleNotFoundError: No module named 'relaykv'`
  になった

## 今日の対応

`run_tiered_kv.py` の先頭に path を追加して解消した。

```python
import sys
from pathlib import Path

sys.path.append(str(Path(__file__).resolve().parents[1]))
```

必要なら `run_baseline.py` にも同じ対応を入れておく。
